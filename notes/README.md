# Course Notes

This directory will contain the rendered course notes book for Data 145.

## Status

Course notes are currently in development and will be published as the semester
progresses.

## When Available

The notes will be a comprehensive reference covering all course topics:

- Probability fundamentals
- Random variables and distributions
- Expectation and variance
- Limit theorems
- Statistical inference
- Linear regression

## Format

The notes are built as a Quarto book and will be available as:

- An interactive HTML website (primary format)
- A downloadable PDF (for offline reading)

Check back soon, or see the [course schedule](../schedule.qmd) for updates on
when specific chapters will be available.
