# Demos

This directory contains in-class demonstration notebooks for Data 145.

## Contents

Jupyter notebooks from lecture demonstrations will be posted here. Each demo
is in its own subdirectory with any required data files.

```
demos/
├── demo01-intro/
│   ├── demo01.ipynb
│   └── data/
├── demo02-simulation/
│   └── demo02.ipynb
└── ...
```

## Running the Notebooks

### Option 1: DataHub (Recommended)

1. Download the notebook file
2. Go to [datahub.berkeley.edu](https://datahub.berkeley.edu)
3. Upload the notebook
4. Run the cells

### Option 2: Local Jupyter

If you have Jupyter installed locally, you can run the notebooks directly after
downloading.
