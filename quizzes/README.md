# Quizzes

This directory contains quiz materials for Data 145.

## Contents

Quiz PDFs will be posted here after each quiz is given in class. We typically
post the quiz without solutions first, then add solutions a few days later.

```
quizzes/
├── quiz01/
│   ├── quiz01.pdf           # Quiz (no solutions)
│   └── quiz01-solutions.pdf # Solutions (posted later)
├── quiz02/
│   └── ...
└── ...
```

## Quiz Policy

- Quizzes are given **in class** (usually on Thursdays)
- They are **closed-book** and cover recent material
- Duration: approximately 15 minutes
- Your **lowest 2 quiz scores** are dropped

## Studying

Quizzes are designed to test your understanding of recent lectures. To prepare:

- Review lecture slides and notes
- Work through examples from class
- Make sure you understand the key concepts, not just the procedures

Previous quizzes (posted here) are excellent practice!
