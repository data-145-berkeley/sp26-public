**Spring 2026 | UC Berkeley**

This repository contains public course materials for Data 145. Materials are
posted here as the course progresses.

## Downloading Materials

### Individual Files

Click on any file in the repository and use the "Download" or "Raw" button.

### Entire Repository

To download all materials:

```bash
git clone https://github.com/berkeley-data145/sp26-public.git
```

Or click the green "Code" button above and select "Download ZIP".

### Staying Updated

If you cloned the repository, pull updates regularly:

```bash
cd sp26-public
git pull
```
