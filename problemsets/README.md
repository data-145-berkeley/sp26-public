# Problem Sets

This directory contains problem set materials for Data 145.

## Contents

Problem set source files (`.tex`) and compiled PDFs (`.pdf`) will be posted
here as the course progresses. Each problem set is in its own subdirectory:

```
problemsets/
├── ps01/
│   ├── ps01.tex    # LaTeX source
│   └── ps01.pdf    # Compiled PDF
├── ps02/
│   └── ...
└── ...
```

## Using the LaTeX Source

We provide the LaTeX source files so you can:

- Type your solutions directly in LaTeX
- See how the problems are formatted
- Learn LaTeX by example

If you're new to LaTeX, we recommend [Overleaf](https://www.overleaf.com) as an
easy way to get started.
